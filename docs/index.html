How it works?

Мы используем CSS Custom Properties и цветовое пространство HSL для динамического вычисления цветов темы.

:root {
  --primary-h: 260;
  --primary-s: 80%;
  --primary-l: 40%;
  --primary: hsl( var(--primary-h), var(--primary-s), var(--primary-l) );
}

Так мы уже построили цвет из HSL-компонент. Но как построить целую палитру?Для этого нужно произвести некие вычисления.

:root {
  --primary-400: hsl(
                      calc( var(--primary-h) * var(--h-400) ),
                      calc( var(--primary-s) * var(--s-400) ),
                      calc( var(--primary-l) * var(--l-400) )
                  );
  --primary-500: hsl(
                      calc( var(--primary-h) * var(--h-500) ),
                      calc( var(--primary-s) * var(--s-500) ),
                      calc( var(--primary-l) * var(--l-500) )
                  );
}

Как это использовать

1. Не хочу ничего устанавливать (использование с pure CSS)

Вы можете скачать непосредственно сгенерированную тему, нажав на кнопку "Download theme" выше. И подключить её к вашему сайту любым удобным для вас способом. Например, так:

<link href="./my-theme.css" rel="stylesheet">

Затем, если хотите, создать стиль для второй темы, например:

.secondary-theme {
   --my-theme-h: 190;
   --my-theme-s: 80;
   --my-theme-l: 40;
   --my-theme-contrast-threshold: 50%;
}

Какой-нибудь кнопкой можно применять эту тему:

const changeThemeButton = document.querySelector('button.change-theme-bth');
changeThemeButton.addEventListener('click', () => {
    document.body.classList.toggle('secondary-theme');
});

Как применять тему на своих элементах? Вам будут доступны 14 цветов, а также контрастные к ним цвета. Разберём на примере:

 button {
     background-color: var(--my-theme);
     color: var(--my-theme-contrast);
     &:hover {
         background-color: var(--my-theme-700);
         color: var(--my-theme-700-contrast);
     }
 }

А где же прозрачные цвета? Всё просто: вы можете самостоятельно настраивать прозрачность у цветов.

     background-color: hsla(
                          var(--my-theme-500-h),
                          var(--my-theme-500-s),
                          var(--my-theme-500-l),
                          62% /* alpha */
                       );

2. Хочу использовать SCSS

При использовании SCSS у вас открываются дополнительные возможности по управлению темой.

Установка:

npm install material-theme-creator

Использование:

@import "~material-theme-creator/core";

// Инициализируем root переменные
@include mtc-init();

body {
  // Создадим тему на основе какого-то цвета
  // Для этого передадим:
  //   1. название темы
  //   2. цвет, на основе которого мы будем строить тему
  //   3. порог контрастности темы
  @include mtc-create-variables-from-color('primary', #cc3300, 50%);
}

body.secondary-theme {
  @include mtc-update-theme('primary', #6200ee, 55%);
}

button {
  color: mtc-color-contrast('primary');
  background-color: mtc-color('primary');
  &:hover {
    color: mtc-color-contrast('primary', 700);
    background-color: mtc-color('primary', 700, 92%);
  }
}

3. Использую Angular Material

Вы можете использовать дополнительный модуль, позволяющий перевести темы Angular Material на использование CSS Custom Properties.

3.1 Подключение
@import "~material-theme-creator/ngx-mtc";
@import '~@angular/material/theming';

@include mat-core();
@include ngx-mtc-init();

$primary-map: ngx-mtc-create-theme-map('primary');
$accent-map: ngx-mtc-create-theme-map('accent');
$warn-map: ngx-mtc-create-theme-map('warn');

У нас есть два варианта для создания тем.

3.2  ПРЕДПОЧТИТЕЛЬНО!  Если вы хотите создавать тему из одного цвета

В случае этого способа, каждая тема будет состоять всего из нескольких строк кода

body {
  --is-dark-theme: 1; // Is dark theme? 1 or 0;
  @include ngx-mtc-theme-base(); // Creates base colors

  // Creates theme colors
  @include ngx-mtc-create-variables-from-color('primary', #009688, 38%);
  @include ngx-mtc-create-variables-from-color('accent', #2196f3, 57%);
  @include ngx-mtc-create-variables-from-color('warn', #f44336, 62%);
}

// Creates Angular Material Theme
@include angular-material-theme(
  ngx-mtc-custom-theme(
    mat-palette($primary-map),
    mat-palette($accent-map),
    mat-palette($warn-map)
  )
);

И код второй темы:

body.theme2 {
  --is-dark-theme: 0;
  @include ngx-mtc-update-theme('primary', #142148, 45%);
  @include ngx-mtc-update-theme('accent', #658e14, 50%);
  @include ngx-mtc-update-theme('warn', #750101, 50%);
}

3.3  НЕ РЕКОМЕНДУЕТСЯ!  Если вы просто хотите использовать CSS Custom Properties в Angular Material Themes и использовать существующие Material темы.

Из-за использования миксина angular-material-theme(), это создаёт очень много CSS-кода для каждой темы.

body {
  @include ngx-mtc-create-variables-from-map('primary', $mat-teal);
  @include ngx-mtc-create-variables-from-map('accent', $mat-deep-purple);
  @include ngx-mtc-create-variables-from-map('warn', $mat-red);
}

// Light theme
@include angular-material-theme(
           mat-light-theme(
             mat-palette($primary-map),
             mat-palette($accent-map),
             mat-palette($warn-map)
           )
         );

// Dark theme
@include angular-material-theme(
           mat-dark-theme(
             mat-palette($primary-map),
             mat-palette($accent-map),
             mat-palette($warn-map)
           )
         );

Вторую тему в этом случае рекомендуется выносить в отдельный файл стилей и подключать так:

<link href="./secondary-theme.css" rel="stylesheet">

И код второй темы:

// secondary-theme.scss
body {
  @include ngx-mtc-update-theme('primary', #142148, 45%);
  @include ngx-mtc-update-theme('accent', #658e14, 50%);
  @include ngx-mtc-update-theme('warn', #750101, 50%);
}

// Если, например, главная тема и вторая тема являются
//   светлыми (или тёмным), то на этом можно остановиться.
// Но если, например, главная тема светлая, а вторая тема тёмная,
//   то нужно дописать следующие строки:

// Light theme
@include angular-material-theme(
           mat-light-theme(
             mat-palette($primary-map),
             mat-palette($accent-map),
             mat-palette($warn-map)
           )
         );

// Dark theme
@include angular-material-theme(
           mat-dark-theme(
             mat-palette($primary-map),
             mat-palette($accent-map),
             mat-palette($warn-map)
           )
         );

3.4 Использование

Use standard Angular material mat-color and mat-contrast to extract individual colors from a palette.

button {
  color: mat-contrast($primary-map, 500);
  background-color: mat-color($primary-map, 500);
  &:hover {
    color: mat-contrast($primary-map, 700);
    background-color: mat-color($primary-map, 700, 92%);
  }
}
