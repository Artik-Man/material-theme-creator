Usage

Simple

  1. Create theme from one color 
    
@import "material-theme-creator"; // Name of this file!

$theme: create-theme-map('primary');
:root {
  @include create-color(#1f7ead, 'primary');
  --primary-contrast-threshold: 50%;
}
  2. Use theme 
     
div {
  background: map-get($theme, 500);
  color: map-contrast($theme, 500);
}

Three-color themes

  1. Create three-color theme 
    
@import "material-theme-creator"; // Name of this file!

:root {
  @include set-color-variables(#03a9f4, #00acc1, #f44336);
  @include set-contrast-threshold(45%, 50%, 55%);
}
  2. Use theme 
    
.primary {
  background: mtc-get($primary-map, 500);
  color: mtc-contrast($primary-map, 500);
}
.accent {
  background: mtc-get($accent-map, 500);
  color: mtc-contrast($accent-map, 500);
}
.warn {
  background: mtc-get($warn-map, 500);
  color: mtc-contrast($warn-map, 500);
}

Using with Angular Material

  1. Setup 
    
@import '~@angular/material/theming';
@import "material-theme-creator"; // Name of this file!

:root {
  @include set-color-variables(#03a9f4, #00acc1, #f44336);
  @include set-contrast-threshold(45%, 50%, 55%);
}

// Light theme
@include angular-material-theme(mat-light-theme(mat-palette($primary-map), mat-palette($accent-map), mat-palette($warn-map)));
// Dark theme
//@include angular-material-theme(mat-dark-theme(mat-palette($primary-map), mat-palette($accent-map), mat-palette($warn-map)));
  2. Use theme 
    
.primary {
  background: mat-get($primary-map, 500);
  color: mat-contrast($primary-map, 500);
}
.accent {
  background: mat-get($accent-map, 500);
  color: mat-contrast($accent-map, 500);
}
.warn {
  background: mat-get($warn-map, 500);
  color: mat-contrast($warn-map, 500);
}

Pure CSS

  1. Compile the code (*.scss -> *.css): 
     
@import "material-theme-creator"; // Name of this file!

:root {
  @include set-color-variables(#03a9f4, #00acc1, #f44336);
  @include set-contrast-threshold(45%, 50%, 55%);
}
  2. Use theme 
    
.primary {
  background: var(--primary);
  color: hsl(0, 0%, calc(((var(--primary-l) * 1%) - var(--primary-contrast-threshold, 50%)) * (-100)));
}
.accent {
  background-color: hsla(var(--accent-h), calc(var(--accent-s) * 1%), calc(var(--accent-l) * 1%), .5);
}
.warn {
  color: hsla(calc(var(--accent-h) + 10), calc(var(--accent-s) * 1% + 10%), calc(var(--accent-l) * 1% - 10%), 1);
}
  3. Change theme for element 
    
.teal {
  --primary-h: 170;
  --primary-s: 100%;
  --primary-l: 30%;
  --primary-contrast-threshold: 35%;

  background: var(--primary);
  color: hsl(0, 0%, calc(((var(--primary-l) * 1%) - var(--primary-contrast-threshold, 50%)) * (-100)));
}
