Usage

Simple

  1. Create theme from one color 
    
$theme: create-theme-map('primary');
:root {
  @include create-color(#1f7ead, 'primary');
  --primary-contrast-threshold: 50%;
}
  2. Use your theme 
     
div {
  background: map-get($theme, 500);
  color: map-contrast($theme, 500);
}

Tree-color themes

  1. Create tree-color theme 
    
:root {
  @include set-color-variables(#03a9f4, #00acc1, #f44336);
  @include set-contrast-threshold(45%, 50%, 55%);
}
  2. Use theme 
    
.primary {
  background: map-get($primary-map, 500);
  color: map-contrast($primary-map, 500);
}
.accent {
  background: map-get($accent-map, 500);
  color: map-contrast($accent-map, 500);
}
.warn {
  background: map-get($warn-map, 500);
  color: map-contrast($warn-map, 500);
}
